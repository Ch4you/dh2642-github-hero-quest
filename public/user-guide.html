User Guide

GitHub Hero Quest

Track your team's GitHub activity, create shared goals, and climb the leaderboard together.

1

Sign in with GitHub

GitHub Hero Quest uses GitHub OAuth for authentication. No password is stored — your GitHub account is used directly.

  1. 1Open the app and click Sign in with GitHub on the welcome screen.
  2. 2A GitHub authorization popup will appear. Approve the request.
  3. 3You will be redirected back to the app and signed in automatically.
  4. 4Your GitHub username and avatar will appear in the top-right corner.
Note: Email/password sign-in is not supported. GitHub OAuth is the only login method.
2

Connect a GitHub Repository

After signing in, connect a public GitHub repository to start tracking activity.

  1. 1Click Workspace in the left sidebar (or bottom nav on mobile).
  2. 2Enter the repository URL or owner/repo name in the input field.
  3. 3Click Connect. The app will validate and fetch initial data.
  4. 4Once connected, click Sync in the top header to load the latest GitHub activity.
  5. 5To switch repositories, click the repository name in the header and select from the list.
Tip: Each repository has its own separate workspace — goals, XP rules, and rankings are all stored per repository.
3

Create and Manage Goals

Goals (Quests) let your team set measurable targets for repository activity within a date range.

  1. 1Click Manage goals in the left sidebar.
  2. 2Click Add to open the goal creation form.
  3. 3Fill in a title, choose a metric (Merged PRs, Commits, Open PRs, or Reviewed PRs), set a target value, and pick start and end dates.
  4. 4Optionally set a reward XP amount for completing the goal.
  5. 5Click Save. The goal will appear in the Active tab once its start date arrives.
  6. 6To mark a goal as done early, click the Complete button next to an active goal in the Dashboard.
  7. 7To delete a goal, open its detail view and click Delete.
Tip: Goals are shared across the team. All members connected to the same repository will see the same goals and progress.
4

Dashboard and Leaderboard

The Dashboard gives you a personal overview of your activity. The Leaderboard shows how the whole team compares.

  1. 1Click Dashboard in the sidebar to see your XP, level, and active goals.
  2. 2The XP breakdown shows how your commits, PRs, and reviews contribute to your score.
  3. 3Click Active Goals to see all current team goals and their progress.
  4. 4Click Team ranking in the sidebar to open the Leaderboard.
  5. 5Use the Last 7 days / All time toggle to switch between weekly and overall rankings.
  6. 6Click Sync in the header at any time to refresh data from GitHub.
Tip: Invite teammates by clicking the copy icon next to their name in the Teammates panel — they need to sign in and connect the same repository to appear on the leaderboard.
